# የአይሲቲ ድጋፍ ማሳወቂያ ስርዓት
# ICT Support Notification System

## ባህሪያት (Features)

### 1. ለአይቲ አስተዳዳሪዎች ማሳወቂያዎች (Notifications for IT Admins)
- ✅ ተጠቃሚዎች አዲስ የድጋፍ ጥያቄ ሲያስገቡ ማሳወቂያ ይደርሳል
- ✅ በጎን አሞሌ ላይ የመጠባበቂያ እና በሂደት ላይ ያሉ ጥያቄዎች ቁጥር ይታያል
- ✅ ማሳወቂያው የጥያቄውን ርዕስ፣ ቅድሚያ እና ምድብ ያካትታል

### 2. ለተጠቃሚዎች ማሳወቂያዎች (Notifications for Users)
- ✅ አይቲ አስተዳዳሪ ጥያቄውን ሲቀበል ማሳወቂያ ይደርሳል
- ✅ ሁኔታ ሲቀየር (በሂደት ላይ፣ ተፈትቷል፣ ተዘግቷል) ማሳወቂያ ይደርሳል
- ✅ መፍትሄ ሲሰጥ ማሳወቂያው የመፍትሄውን ማጠቃለያ ያካትታል

### 3. የጎን አሞሌ ባጅ (Sidebar Badge)
- ✅ "💻 አይሲቲ ድጋፍ" አገናኝ ላይ ቁጥር ይታያል
- ✅ የመጠባበቂያ (pending) እና በሂደት ላይ (in_progress) ያሉ ጥያቄዎችን ይቆጥራል
- ✅ ብርቱካናማ ቀለም (orange badge) ለአይቲ አስተዳዳሪዎች ብቻ ይታያል

## የስራ ሂደት (Workflow)

### ተጠቃሚ የድጋፍ ጥያቄ ሲያስገባ:
1. ተጠቃሚ ቅጹን ይሞላል እና "ጥያቄ ላክ" ይጫናል
2. ጥያቄው በዳታቤዝ ውስጥ ይቀመጣል (status: pending)
3. **ሁሉም አይቲ አስተዳዳሪዎች ማሳወቂያ ይደርሳቸዋል**
4. በጎን አሞሌ ላይ ባጅ ቁጥር ይጨምራል

### አይቲ አስተዳዳሪ ስራ ሲጀምር:
1. አይቲ አስተዳዳሪ "🚀 ስራ ጀምር" ይጫናል
2. ሁኔታ ወደ "in_progress" ይቀየራል
3. **ተጠቃሚ ማሳወቂያ ይደርሰዋል**: "የአይሲቲ ድጋፍ ጥያቄዎ በሂደት ላይ ነው"
4. አይቲ አስተዳዳሪ ወደ ቦታው ሄዶ ችግሩን ይፈታል

### አይቲ አስተዳዳሪ መልስ ሲሰጥ:
1. አይቲ አስተዳዳሪ "💬 መልስ ስጥ" ይጫናል
2. መፍትሄውን ይጽፋል እና ሁኔታ ይመርጣል (resolved/closed)
3. **ተጠቃሚ ማሳወቂያ ይደርሰዋል** ከመፍትሄው ማጠቃለያ ጋር
4. በጎን አሞሌ ላይ ባጅ ቁጥር ይቀንሳል

## ቴክኒካል ዝርዝሮች (Technical Details)

### የተሻሻሉ ፋይሎች:
1. `views/components/sidebar.php` - የባጅ ቆጣሪ ታክሏል
2. `public/ict-support.php` - ማሳወቂያ ሲፈጠር ይላካል
3. `public/api/ict-support/respond.php` - ማሳወቂያ ሲመለስ ይላካል

### የዳታቤዝ ጥያቄዎች:
```sql
-- የመጠባበቂያ እና በሂደት ላይ ያሉ ጥያቄዎችን ቁጥር
SELECT COUNT(*) 
FROM ict_support_requests 
WHERE status IN ('pending', 'in_progress')
```

### PHP ኮድ:
```php
// ለአይቲ አስተዳዳሪዎች ማሳወቂያ መላክ
NotificationService::notifyDepartment(DEPT_IT_ADMIN, null, $message);

// ለተጠቃሚ ማሳወቂያ መላክ
NotificationService::notify($request['requester_id'], null, $message);
```
